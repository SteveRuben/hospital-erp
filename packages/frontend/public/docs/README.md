# Screenshots Documentation

Place your screenshots in this folder with the following naming convention:

- `dashboard.png` — Dashboard principal
- `patients-list.png` — Liste des patients
- `patient-detail.png` — Détail d'un patient
- `consultation.png` — Formulaire de consultation
- `laboratoire.png` — Vue Kanban du laboratoire
- `finances.png` — Page finances
- `facturation.png` — Facturation avec facture
- `pharmacie.png` — Gestion de la pharmacie
- `rdv.png` — Rendez-vous
- `file-attente.png` — File d'attente
- `lits.png` — Gestion des lits
- `portail.png` — Portail patient
- `login.png` — Page de connexion
- `landing.png` — Landing page

These will be referenced in the Documentation page at `/app/documentation`.
